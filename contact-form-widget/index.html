📨 Виджет «Обратная связь»

Лёгкая всплывающая форма для любого сайта. Не нужны библиотеки — всё на чистом HTML/CSS/JS.

Что внутри:

  • Плавающая кнопка-FAB с анимацией пульсации
  • Многошаговая модальная форма (контакты + детали проекта)
  • Email, телефон с маской, выбор услуги, бюджет, срочность
  • JS-валидация в реальном времени с зелёной галочкой
  • Анимированное состояние «успешно отправлено» с номером заявки
  • Закрытие по клику вне окна и по клавише Esc
  • История отправленных заявок (sessionStorage)
  • Настройка FAB: пульсация, позиция, цвет
  • Полностью адаптивный

Нажмите фиолетовую кнопку в правом нижнем углу — увидите виджет в действии.
